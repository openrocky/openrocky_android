## App Name

## Full Description (4000 chars max)
Rocky is a voice-first AI agent that goes beyond simple chatbots. It's not a chat wrapper — it's a fully integrated AI assistant that can interact with your device, execute tasks, and deliver results through voice or text.

🎙️ VOICE-FIRST DESIGN
Talk naturally with Rocky using real-time voice providers (OpenAI Realtime, Gemini Live, Doubao). Rocky listens, plans, executes tools, and speaks back — all in real-time.

🔧 30+ NATIVE TOOLS
Rocky can interact with your Android device directly:
• Weather forecasts (Open-Meteo)
• Calendar events & reminders
• Contacts search
• Location & nearby places
• File management & workspace
• Web search & browser
• Notifications & alarms
• Shell command execution
• Embedded Python 3.11 interpreter
• Cryptographic operations
• Email compose
• And more...

🤖 10 AI PROVIDERS
Connect to your preferred AI provider:
• OpenAI (GPT-4o, GPT-5)
• Anthropic (Claude)
• Google Gemini
• DeepSeek
• Groq, xAI, OpenRouter
• Volcengine Doubao
• Azure OpenAI
• AIProxy
All with custom host support for self-hosted deployments.

🎭 5 AI CHARACTERS
Switch between personalities:
• Rocky — friendly, efficient assistant
• English Teacher — patient language tutor
• Software Dev Expert — technical engineer
• Storm Chaser — weather enthusiast
• Mindful Guide — wellness coach
Create your own custom characters too!

⚡ 10 BUILT-IN SKILLS
• Translator, Summarizer, Writing Coach
• Code Helper, Math Solver
• Travel Planner, Health Insights
• Daily Briefing, Research Assistant
• Quick Convert
Import and create custom skills!

🧠 PERSISTENT MEMORY
Rocky remembers across sessions. Key-value memory lets Rocky recall your preferences, habits, and context.

🐍 EMBEDDED PYTHON
Run Python code directly on your device with a full standard library — no setup needed. Great for calculations, data processing, and automation.

🔒 SECURE & PRIVATE
• All API keys stored in encrypted storage
• No data collection — your conversations stay on your device
• Open source

Rocky is open source under the OpenRocky project. Visit github.com/openrocky for more.
